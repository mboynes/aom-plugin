This plugin adds a magicians directory to the site, adds shortcodes for rendering links to a Magician, and more. As seen in _Use Your Allusion_.

== Installation ==

1. Upload to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
